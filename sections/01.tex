
\section{引言与基础文本}
\label{sec:introduction}
本节展示了基本的段落文本和一些文本格式。我们强调一些\textbf{重要}的词汇，以及一些\textit{斜体}的表达。您也可以使用\texttt{等宽字体}来表示代码或命令。

\subsection{列表示例}
我们可以创建有序列表和无序列表来组织信息。

\subsubsection*{无序列表}
\begin{itemize}
    \item 第一个项目：关于文本结构。
    \item 第二个项目：关于数学公式的展示。
    \item 第三个项目：关于图表的占位符。
\end{itemize}

\subsubsection*{有序列表}
\begin{enumerate}
    \item 第一步：定义文档结构。
    \item 第二步：插入核心内容。
    \item 第三步：编译并检查结果。
\end{enumerate}
